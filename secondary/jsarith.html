JavaScript Arithmetic


JavaScript's logic allows the use of typical arithmetic characters within the JavaScript console. It allows you to use basic math equations, including the ability to create variables. The language uses Boolean logic, as well.


Some examples of JavaScript arithmetic in action:

  • 2 * 6 ;
  • 12
  • var x = 25;
  • undefined
  • x + 4;
  • 29
  • 6 * 9 + 4;
  • 58
  • 6 * (9 + 4);
