JavaScript Variables


JavaScript's logic allows the use of typical arithmetic characters within the JavaScript console, including variables. Variables allow the user to store custom data values, in the form of "strings", numericals, and arrays. (For more information on strings and arrays, please visit our other pages!) The language uses Boolean logic upon the variables, as well.


Some examples of JavaScript variables in action:

  • var name = "mike";
  • undefined
  • name;
  • "mike"
  • var favoriteNumber = 17;
  • undefined
  • favoriteNumber;
  • 17
  • favoriteNumber = 34;
  • 34
  • favoriteNumber;
  • 34
  • var num1 = 7;
  • undefined
  • var num2 = 9;
  • undefined
  • var num3 = num1 + num2;
  • undefined
  • num3;
  • 16
